# UI/UX Design Intelligence

Source: ui-ux-pro-max plugin (v2.5.0)

## Quick style picks for hackathon
| Vibe | Style | Use for |
|------|-------|---------|
| Modern SaaS | Glassmorphism + dark | Dashboard, admin |
| Clean startup | Minimalism | Landing page |
| Bold/viral | Brutalism | Portfolio, showcase |
| Soft app | Claymorphism | Mobile, consumer |
| Bento grid | Bento grid | Feature showcase |

## Color palette strategy
- SaaS/tech → blue/purple/slate
- Health/wellness → green/teal
- Finance → navy/gold
- Creative → vibrant gradients

## Font pairing formula
- Heading: Inter / Geist / Plus Jakarta Sans
- Body: Inter / DM Sans
- Code: JetBrains Mono / Fira Code

## Invoke
```
/ui-ux-pro-max build [component] in [style] for [product type]
/design [what you need]
```

## Accessibility baseline
- Contrast ratio 4.5:1 minimum
- Focus visible on all interactive elements
- Screen reader labels on icons
